# 🎨 NanoBanana プロンプトシート生成スキル

Google Sheetsのストーリーボード（またはローカルCSV）から、**NanoBanana用のプロンプトだけを抽出**して、コピペしやすいテキスト形式で出力する [Antigravity](https://antigravity.dev) スキルです。

## 📦 インストール

1. このリポジトリをcloneまたはダウンロード
2. `SKILL.md` を Antigravity の skills ディレクトリに配置

```
~/.gemini/antigravity/skills/nanobanana-prompt-sheet/
└── SKILL.md
```

## 🚀 使い方

Antigravity に以下のように話しかけるだけ：

### Google Sheets から抽出

```
このスプシからプロンプトシート作って
https://docs.google.com/spreadsheets/d/XXXXX/edit?gid=YYYYYY
```

### ローカルCSV から抽出

```
このCSVからプロンプトシート作って
```

### トリガーキーワード

- 「プロンプトシート」
- 「NanoBananaプロンプト抽出」
- 「スプシからプロンプト」
- 「プロンプト一覧」

## 📄 出力フォーマット

各カットごとに以下のブロックが出力されます：

```
────────────────────────────────────────────────────────────
【カット1】P0:フック
────────────────────────────────────────────────────────────

▼ 台本（参考）
YouTubeを始めた人の9割は、3ヶ月以内に消えます。

▼ プロンプト パターン1（↓これをコピペ）
暗闇の中から赤く光る「9割が消える」という巨大な文字が...（省略）...16:9

▼ プロンプト パターン2（↓これをコピペ）
崩壊するYouTube再生ボタンのアイコン...（省略）...16:9
```

**▼ の横のプロンプトをそのままNanoBananaに貼り付けるだけ**で使えます。

## 📋 前提条件

- [Antigravity](https://antigravity.dev) がインストールされていること
- Node.js が利用可能であること（CSVパース用スクリプトの実行に使用）
- Google Sheets を使う場合、シートが「リンクを知っている全員が閲覧可」に設定されていること

## 🗂 対応するスプレッドシート列構成

[ストーリーボード生成スキル](https://github.com/) が出力するCSVの列構成に対応：

| 列 | 内容 | 使用 |
|---|---|---|
| A | カット番号 | ✅ |
| B | セクション | ✅ |
| C | (タイム) | — |
| D | ナレーション | ✅ |
| E〜G | (素材系) | — |
| H | NanoBanana パターン1 | ✅ |
| I | NanoBanana パターン2 | ✅ |
